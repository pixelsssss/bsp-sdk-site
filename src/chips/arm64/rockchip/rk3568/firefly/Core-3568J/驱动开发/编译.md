## 1. 编译配置

将 bsp 导入 ide 工作空间后，需要右键选择工程属性，配置依赖的 base 工程，如下表格：

| 操作系统位数 | base 选项                                                    |
| ------------ | ------------------------------------------------------------ |
| 64 位        | **Toolchain**: aarch64-sylixos-toolchain**CPU** **Type**: generic**FPU Type**: default |
| 32 位        | **Toolchain**: arm-sylixos-toolchain**CPU** **Type**: cortex-a57**FPU Type**: disable |
